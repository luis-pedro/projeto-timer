1ª, 2ª e 3ª Versão

As 3 primeiras versões foi importante na construção do site, aqui começou as primeiras páginas e também as primeiras estilizações

[MODELO BÁSICO]

1ª Versão - O início

A versão mais importante pois aqui a ideia foi tirada do pápel

Foram feitas configurações como:

  1. Criação do HTML básico
  2. Criação dos inputs (Timer, Pausas, Duolingo, Descanço)
  3. Primeira estilização

2ª Versão - Planos

Aqui o projeto começou a caminhar e ser mais projetado, ideias foram surgindo como a da loja que será criada apenas mais para frente

Alterações feitas:

  1. Criação das páginas (Timer, Perfil e Loja)
  2. Definindo as primeiras cores iniciais do projeto
  3. Iniciado o protótipo de como vai ficar a tela de perfil
  4. Foi adicionado o input para definir as tarefas que será realizado nos timers
  5. Foi feita uma inversão do descanso do 1 timer

3ª Versão - O passo adiante

Acabando as versões iniciais, a 3ª versão foi a mais importante tanto em questão de ideias quanto na estilização e criação do HTML, aqui foi dado o passo para vir a primeira linguagem de programação (o Java Script)

Alterações Feitas:

  1. Alteração da paleta de cores para cores mais cleans e agradáveis

    2. Cores usadas:

    Branco -> #ECEBE4

    Preto -> #0C0910

    Azul -> #429EA6

    Laranja -> #FF7F11

  2. Alteração do layout do projeto
  3. Alteração do input "Duolingo" para "extras" (abaixando o tempo de 15 parra 10 minutos)
  4. Adição do input "horas estudadas" ( Mais para frente será usado Java Script para fazer a contagem)
  5. Adição do aviso nas telas de perfil e sobre nós ( modelo básico disponível na 4ª versão )
  6. Criação e estilização do modelo básico da tela trragetória que será apresentado todas as modificações do site
